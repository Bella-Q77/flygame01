# FlyGame01 前端展示页面

这是一个为FlyGame01飞行射击游戏开发的高质量前端展示页面，包含游戏介绍、特色功能展示、截图轮播、系统需求说明等核心板块。

## 项目结构

```
├── index.html      # 主页面文件
├── styles.css      # 样式文件
├── script.js       # JavaScript交互文件
└── README.md       # 项目说明文件
```

## 功能特点

### 1. 响应式设计
- 适配不同屏幕尺寸，包括桌面端、平板和移动端
- 使用现代CSS Grid和Flexbox布局
- 流畅的动画和过渡效果

### 2. 核心板块
- **首页横幅**：游戏标题、简介和CTA按钮
- **游戏介绍**：游戏背景、目标和基本玩法
- **特色功能**：展示游戏的主要特色和功能
- **游戏截图**：自动轮播的游戏画面展示
- **系统需求**：最低配置和推荐配置说明
- **立即体验**：游戏控制说明和下载链接

### 3. 交互功能
- 平滑滚动导航
- 截图自动轮播和手动切换
- 滚动动画效果
- 移动端汉堡菜单
- 键盘导航支持
- 图片懒加载

## 使用方法

### 直接打开

1. 下载项目文件
2. 双击打开 `index.html` 文件
3. 在浏览器中查看页面

### 使用本地服务器

```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 使用Node.js启动本地服务器
npx serve .

# 然后在浏览器中访问 http://localhost:8000
```
